# Archive

This directory is reserved for historical implementation notes, one-off audit
outputs, migration snapshots, and older planning material.

Current operator and contributor authority lives in:

- `AGENTS.md`
- `README.md`
- `ARCHITECTURE.md`
- `API_DOCUMENTATION.md`
- `DEPLOYING.md`
- `docs/DEPLOYING.md`
- `docs/OPERATIONS.md`
- `CONTRIBUTING.md`
- `SECURITY.md`

Historical root-level snapshots were removed from the public root to keep the
GitHub landing page focused. Use git history when an older implementation note
is needed for archaeology.
